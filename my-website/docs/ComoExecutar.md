# Como Executar a Aplicação

## Introdução

Esta secção descreve os **passos necessários para executar a aplicação**, desde os pré-requisitos até à utilização do menu principal.

A aplicação foi desenvolvida para funcionar exclusivamente em **modo consola**, não sendo necessária qualquer interface gráfica ou framework web.

---

## Pré-requisitos

Antes de executar a aplicação, é necessário garantir que o sistema possui:

- **Python 3.10 ou superior**
- Sistema operativo com suporte a terminal (Windows, Linux ou macOS)
- Git (opcional, apenas para clonagem do repositório)

Para verificar a versão do Python instalada:
```bash
python --version
